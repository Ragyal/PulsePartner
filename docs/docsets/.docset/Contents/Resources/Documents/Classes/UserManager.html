  • logout()

    Signs an user out via FirebaseAuth. Removes an existing FCM-Token and updates the document in Firestore. The Firestore update triggers an cloud function to delete existing MatchData of that user. Finally the application unregisters from remote notifications and returns to LoginViewController.
